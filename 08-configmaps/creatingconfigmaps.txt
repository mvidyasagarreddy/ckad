Creating config maps has two steps

1.creating config maps
2.Injecting the configmap into pod.

There are two ways to create a configmap

1.imperative
2.declarative

imperative:

kubectl create configmap <config-name> --from-literal=<key>=<value>

kubectl create configmap app-config --from-literal=APP_COLOR=blue --from-literal=APP_MOD-prod

kubectl create configmap app-config --from-file=<path-to-file>

kubectl create configmap app-config --from-file=app_config.properities.
